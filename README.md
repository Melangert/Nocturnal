Nocturnal is a simple ASCII art displayer of famous night time paintings like Starry night in your terminal

<img width="1914" height="927" alt="image" src="https://github.com/user-attachments/assets/55633494-3445-4e1d-a32a-1bf25326a1e4" />


## Usage
run node run.js "painting name" in the Nocturnal directory

## Available paintings
starry night

starry night over the rhone

cafe terrace at night

whistlers nocturne
